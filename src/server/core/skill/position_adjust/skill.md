## 调仓工作流

### Step 1: 收集信息

调用 `position_adjustment_advice`（无需 input），该工具会向用户展示表单，收集：

- 总资产、投资目标、风险偏好、投资经验
- 当前持仓（股票/基金/债券/贵金属/现金）
- 市场温度（1-10）、个人情绪
- 目标仓位、止损止盈设置、流动性需求

### Step 2: 等待用户提交

用户填写并提交表单后，工具会自动调用 LLM 生成调整建议。

### Step 3: 解读结果

工具返回 `{ submitted, advice }`：

- `submitted: false` — 用户取消了表单
- `submitted: true, advice: "..."` — 根据建议内容，给用户清晰的总结，强调操作方向和风险提示

## 注意事项

- 不要重复询问表单中已有的信息
- 不提供具体股票代码推荐
- 始终强调风险管理
- 建议仅供参考，不构成投资建议
